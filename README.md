# 两天搞定人工智能毕业设计之【语义分割】

语义分割和图像分割是人工智能、机器学习、深度学习和计算机视觉解决的核心任务之一。

在医疗影像分析、遥感图像分析、工业缺陷检测、无人驾驶感知等领域广泛落地应用。

同济子豪兄手把手带你使用OpenMMLab开源语义分割算法库MMSegmentation，跑通语义分割开发全流程：标注、训练、评估、预测、部署。
